This project analyzes airline data using Python to clean, process, and visualize key business metrics.

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Files Included
- 01_data_cleaning_airline.ipynb – Data cleaning and preprocessing
- 02_visualization_airline.ipynb – Visual analysis and insights

## Key Outcomes
- Improved data quality through cleaning
- Clear visual insights into airline performance
